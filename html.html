Dart Cheat Sheet [HTML]

Query the DOM

  • query(cssSelector);
  • queryAll(cssSelector);
  • query('#foo');
  • queryAll('div');
  • queryAll('[name="foo"]');
  • queryAll('.foo');
  • query('.foo .bar');
  • queryAll('.foo .bar');

HTTP Request

HttpRequest.getString('http://theUrl').then((response) {
  doSomethingWith(response);
});
